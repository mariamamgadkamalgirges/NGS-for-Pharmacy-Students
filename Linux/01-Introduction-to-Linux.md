# Introduction to Linux

## What is Linux?

Linux is an open-source operating system widely used in Bioinformatics, genomics, and scientific research.

## Why do Bioinformaticians use Linux?

- Fast and efficient
- Handles large sequencing data
- Supports powerful command-line tools
- Runs most bioinformatics software

## Common Linux Distributions

- Ubuntu
- Debian
- CentOS
- Fedora
