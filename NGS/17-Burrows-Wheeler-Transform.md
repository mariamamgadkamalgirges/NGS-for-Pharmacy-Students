# Burrows-Wheeler Transform (BWT)

## What is BWT?

The Burrows-Wheeler Transform is a data transformation algorithm used for efficient sequence alignment.

## Why is BWT Important?

- Compresses sequence data
- Creates efficient genome indexes
- Improves alignment accuracy

## Used In

- BWA
- Bowtie
- Bowtie2

## Advantages

- High alignment accuracy
- Low memory usage
- Suitable for large genomes

## Limitation

Building the index takes time.
