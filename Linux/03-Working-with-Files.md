# Working with Files

## View File Content
cat file.txt

Display the content of a file.
less file.txt

View a file page by page.

## Search Inside Files
grep "word" file.txt

Search for a specific word inside a file.

## Count Lines, Words, and Characters
wc file.txt

## Find Files
find . -name "*.fastq"

Find all FASTQ files in the current directory.

## Display Current Directory
pwd

## List Hidden Files
ls -a
