# Genome Assembly vs Alignment

## Genome Assembly

Genome assembly is the process of combining short sequencing reads into longer DNA sequences (contigs or scaffolds).

### When is Assembly Used?

- No reference genome is available.
- De novo sequencing projects.
- Genome reconstruction.

## Genome Alignment

Genome alignment is the process of mapping sequencing reads to a known reference genome.

### When is Alignment Used?

- A reference genome is available.
- Whole Genome Sequencing (WGS)
- Variant calling
- Gene expression analysis

## Key Difference

| Assembly | Alignment |
|----------|-----------|
| No reference genome | Reference genome required |
| Builds a genome | Maps reads to a genome |
| Used in de novo sequencing | Used in re-sequencing |
