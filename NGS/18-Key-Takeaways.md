# Key Takeaways

## Genome Assembly

- Builds genomes from sequencing reads.
- Used when no reference genome is available.

## Genome Alignment

- Maps reads to a reference genome.
- Used in re-sequencing projects.

## Important Concepts

- Quality Control (QC)
- Assembly Algorithms
- De Bruijn Graph
- OLC Assembly
- Hybrid Assembly
- Metagenomics
- Binning
- Hash Tables
- Burrows-Wheeler Transform (BWT)
